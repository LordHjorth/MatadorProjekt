*** README *** 

Two ways to create the database:

1. Option.

	-> Right click on ‘MatadorProjektSQL.sql’.

	-> Open with… 

	-> Search for ‘MySql Workbench’ (or similar)

	-> Execute query.


2. Option.

	Open ‘MatadorProjektSQL.sql’ and copy all the content.

	Open workbench (or similar) and insert the content to an empty SQL 	tab.

	Execute the whole statement. 

	The database is now created with the necessary tables and data. 

	The database is ready. You can now start the game.
